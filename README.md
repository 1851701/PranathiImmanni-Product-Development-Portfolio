## About me

I'm **Pranathi Immanni** — a **product designer** who cares about how people think, feel, and move through digital spaces. I build **human-centered UX** with clear hierarchy, accessible patterns, and interfaces that stay calm when life is loud.

I’m pursuing a **B.S. in Cognitive Science** at the **University of California, Irvine** (**3rd year**). My coursework in perception, attention, and human-centered research still shapes how I frame problems: **evidence over assumption**, **clarity over noise**.

Born and raised in **San Diego, CA** — balance, light, and open space show up in how I structure layouts on screen.

<details>
<summary><strong>Developer setup — run this project locally</strong></summary>

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

**Production build**

```bash
npm run build
npm start
```

**Customize content**

| What | Where |
|------|--------|
| Name, bio, skills, experience, social links | `src/lib/site.ts` |
| Projects and case studies | `src/data/projects.ts` |
| Resume text | `src/components/resume/resume-section.tsx` |
| PDF resume | Add `public/resume.pdf` |
| Site URL (SEO, sitemap) | `.env.local` → `NEXT_PUBLIC_SITE_URL` (see `.env.example`) |

**Features:** Home, About, Projects, case studies, Resume, Contact · dark/light theme · Framer Motion · contact API route · `sitemap.xml` / `robots.txt`

Placeholder images use [picsum.photos](https://picsum.photos/); swap for your assets in `projects.ts` or `public/`.

</details>
